# Legacy Scripts

These scripts are intentionally kept out of the main product path.

- `image_prep/` contains older still-image prep helpers that predate the current `outpaint_images.py` workflow.
- `video_generation/` contains earlier manual Kling generators that were superseded by `generate_all_videos.py`.

They still point at the project root data folders, but they are archived so the active workflow stays easy to scan.
